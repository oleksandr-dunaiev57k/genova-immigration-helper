# Genova Immigrant Helper 🇮🇹
Step-by-step guide for Ukrainian immigrants in Genova, Italy - documents, deadlines, community 

A simple tool to help Ukrainian immigrants in Genova navigate Italian bureaucracy: 
permesso di soggiorno, codice fiscale, tessera sanitaria, residenza, school enrollment.

## Why this exists
I moved from Kharkiv, Ukraine to Genova in 2022. Getting documents was confusing 
for my family - and it's confusing for thousands of others. This project documents 
the real steps, with practical tips, in plain language.

## Status
🔨 Currently in MVP phase. Website live with AI chatbot on Vercel.
